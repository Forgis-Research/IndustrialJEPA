# Experiment Figures

Store all visualizations here with accompanying documentation.

## Naming Convention

```
{experiment_number}_{description}.png
{experiment_number}_{description}.md
```

## Required Figures

- [ ] Distribution comparison (source vs target)
- [ ] Transfer learning curves
- [ ] Anomaly score distributions
- [ ] ROC curves
- [ ] Embedding visualizations (t-SNE/UMAP)
- [ ] Confusion matrices

## Template

For each figure, create an accompanying markdown file:

```markdown
# [Title]

![](./filename.png)

## What it shows
[Description]

## Key observations
- Observation 1
- Observation 2

## Implications for research
[What this means]
```
