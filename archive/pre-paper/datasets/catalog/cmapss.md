# C-MAPSS / N-CMAPSS (NASA Turbofan Engine Degradation)

## Executive Summary
- **Domain**: Aerospace / Turbomachinery
- **Task**: Remaining Useful Life (RUL) prediction; adaptable to sensor forecasting
- **Size (C-MAPSS)**: 4 sub-datasets, 436–874 units, 128–562 cycles, 21 sensors + 3 operational settings
- **Size (N-CMAPSS)**: 8 sub-datasets, larger scale, 18 virtual sensors + 14 health parameters
- **Sampling Rate**: 1 sample per flight cycle (not Hz-based)
- **Real vs Synthetic**: Synthetic — NASA MAPSS (Modular Aero-Propulsion System Simulation)
- **License**: Public domain (NASA)
- **Download URL (C-MAPSS)**: https://data.nasa.gov/Aerospace/CMAPSS-Jet-Engine-Simulated-Data/ff5v-kuh6
- **Download URL (N-CMAPSS)**: https://data.nasa.gov/dataset/N-CMAPSS/
- **Published SOTA**: Extensive for RUL; limited for sensor forecasting

## Detailed Description

C-MAPSS (Commercial Modular Aero-Propulsion System Simulation) generates run-to-failure trajectories for turbofan engines under different fault modes and operating conditions. Despite being widely used, it is fundamentally a simulation.

### C-MAPSS Sub-Datasets

### Known Issues / Limitations
1. **Synthetic** — generated by MAPSS simulation, not real sensor data
2. **Constant within cycle** — 14 of 21 sensors are effectively constant (nearly zero variance)
3. **Physics ≈ random mask** — Exp 48 found p=0.528 (physics mask not better than random)
4. **Task mismatch** — Standard task is RUL, not forecasting; non-standard reformulation has no SOTA
5. **Correlated degradation** — all sensors degrade together (single failure mode in FD001/FD002), limiting group independence

## Relevance to IndustrialJEPA

### Current Usage
Used as Tier 2 in the Rapid Evaluation Suite (experiments 7–20, 43, 48). Key finding: physics mask ≈ random mask (p=0.528), unlike the double pendulum where physics mask is clearly better.

### Physics Grouping Assessment
**Poor fit** — sensors are physically grouped but statistically correlated (all degrade together). This is why the physics mask doesn't help: the independence assumption of grouping doesn't hold.

### Verdict
**Keep as Tier 2-synthetic** for the paper (demonstrates the "correlated system" case where physics ≈ random), but do NOT claim it as the "real mechanical system" Tier 2. The paper story needs a real-data Tier 2.
