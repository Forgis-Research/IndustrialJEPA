1. C-MAPSS RMSE Probe

Frozen-encoder MSE probe on C-MAPSS. The encoder is pretrained via JEPA, then frozen; a small MLP head is trained to predict RUL in cycles. This tests whether FAM’s self-supervised representations encode useful RUL information without any surface-based evaluation.

STAR SOTA reference values (from literature): FD001 = 11.25, FD002 = 18.86, FD003 = 11.37.

Code 
# STAR SOTA RMSE values (literature)
STAR_SOTA = {'FD001': 11.25, 'FD002': 18.86, 'FD003': 11.37}

datasets = ['FD001', 'FD002', 'FD003']
lf_labels = ['100%', '10%']
lf_keys = ['lf100', 'lf10']

fig, axes = plt.subplots(1, 3, figsize=(14, 4.5), sharey=False)

for i, ds in enumerate(datasets):
    ax = axes[i]
    sota = STAR_SOTA[ds]

    vals = []
    stds = []
    for lf in lf_keys:
        key = f'{ds}_{lf}'
        vals.append(rmse_agg[key]['rmse_mean'])
        stds.append(rmse_agg[key]['rmse_std'])

    x = np.arange(len(lf_labels))
    width = 0.3

    # SOTA line
    ax.axhline(y=sota, color=C_SOTA, linestyle='--', linewidth=1.2, zorder=1)
    ax.text(1.05, sota, f'STAR\n{sota:.1f}', va='center', fontsize=8,
            color=C_SOTA, fontweight='bold')

    # FAM bars
    bars = ax.bar(x, vals, width * 2, yerr=stds, capsize=4,
                  error_kw=dict(elinewidth=0.8),
                  color=[C_FAM, C_FAM], edgecolor='white', linewidth=0.5, zorder=2)

    # Color-code by gap closure
    for j, (v, bar) in enumerate(zip(vals, bars)):
        # Gap closure = how much of (FAM - SOTA) gap is closed relative to a "bad" baseline
        # Simple: green if within 2x SOTA, red if > 2x
        gap_ratio = v / sota
        if gap_ratio < 1.8:
            bar.set_facecolor(C_GOOD)
        else:
            bar.set_facecolor(C_BAD)

        ax.text(bar.get_x() + bar.get_width() / 2, v + stds[j] + 0.5,
                f'{v:.1f}', ha='center', va='bottom', fontsize=9, fontweight='bold')

    ax.set_xticks(x)
    ax.set_xticklabels(lf_labels)
    ax.set_title(ds, fontsize=12, fontweight='bold')
    ax.set_ylabel('Test RMSE (cycles)' if i == 0 else '')
    ax.set_ylim(0, max(vals) * 1.35)

fig.suptitle('C-MAPSS RMSE Probe: FAM Frozen Encoder vs STAR SOTA', fontsize=13, fontweight='bold', y=1.02)
plt.tight_layout()

3. PA-F1 vs non-PA F1 Gap: The Protocol Mismatch Trap

Point-adjust (PA) F1 inflates scores by 2-5x because it credits the model for detecting any point within an anomaly segment. Non-PA F1 is the honest metric.

Code

for ds in anomaly_datasets:
    key = f'{ds}_lf100'

        a = legacy[key]['agg']
        pa = a.get('best_pa_f1_mean', np.nan)
        npa = a.get('best_non_pa_f1_mean', np.nan)
        if not np.isnan(pa) and not np.isnan(npa):
            ds_list.append(ds)

4. GECCO/BATADAL Deep-Dive

Both GECCO and BATADAL show decent AUROC but low non-PA F1. The root cause is extreme class imbalance: GECCO has <0.7% positives at short horizons.

Code 
fig, axes = plt.subplots(2, 3, figsize=(16, 8))

for row, ds_name in enumerate(['GECCO', 'BATADAL']):
